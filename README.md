Key Features:

🏆 Tournament Overview

IPL title winners
Total tournament sixes and fours
Season-wise filtering

🏏 Player Performance Analysis

Orange Cap winner (highest run scorer)
Purple Cap winner (highest wicket taker)
Total runs, strike rate, boundaries, and sixes

📊 Match Insights

Toss winner and toss decision analysis
Match result breakdown (runs, wickets, super overs, no results)
Team-wise total wins across seasons

🏟️ Venue Analysis

Matches won by venue
Venue-wise result patterns and performance metrics

🎯 Interactive Visualizations

Dynamic slicers and filters
Doughnut charts, bar charts, KPI cards, and tables
User-friendly interface for exploring IPL statistics

Technologies Used:

Power BI – Dashboard creation and visualization
Excel / CSV – Data source and preprocessing
DAX – Measures and calculated fields
Power Query – Data cleaning and transformation
